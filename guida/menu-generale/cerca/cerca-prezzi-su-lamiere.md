# Cerca Pezzi su Lamiere

Ricerca i pezzi sulle lamiere per mezzo di filtri di selezione.

<u>I filtri di selezione sono:</u>

| Filtro | Descrizione     |
| ------ | --------------- |
| Order  | Numero commessa |
| Phase  | Fase            |
| Name   | Nome contorno   |
