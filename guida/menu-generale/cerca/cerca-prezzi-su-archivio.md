# Cerca Pezzi su Archivio

Ricerca i pezzi sull'archivio del disegno per mezzo di filtri di selezione.

<u>I filtri di selezione sono:</u>

| Filtro | Descrizione     |
| ------ | --------------- |
| Order  | Numero commessa |
| Phase  | Fase            |
| Name   | Nome contorno   |
